# ⚛️ React Machine Coding

A collection of **React machine coding problems and UI challenges** solved using modern React concepts.
This repository is built to **practice frontend machine coding rounds**, strengthen component design skills, and prepare for **frontend interviews**.

The project contains **multiple small React challenges** implemented with clean component architecture, reusable logic, and scoped styling using **CSS Modules**.

---

# 📌 Purpose of This Repository

Frontend interviews often include **machine coding rounds** where developers must build UI features in a limited time.

This repository helps practice:

* Component architecture
* React Hooks
* State management
* Event handling
* UI logic implementation
* Reusable components
* Clean folder structures
* Scoped styling with CSS Modules

The goal is to **improve speed, thinking, and implementation skills for real frontend interview scenarios.**

---

# 🚀 How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/rydhammahajan/react-easy-machine-coding.git
```

---

### 2️⃣ Navigate to Project Folder

```bash
cd react-easy-machine-coding
```

---

### 3️⃣ Install Dependencies

```bash
npm install
```

---

### 4️⃣ Start the Development Server

```bash
npm start
```

---

The application will run on:

```
http://localhost:1234
```
